# VOYAGE — Curated Travel Website

A single-page luxury travel site inspired by the FLEUR aesthetic: warm neutrals, arch motifs, **Fraunces** + **Plus Jakarta Sans** typography, GSAP scroll animations, and a **light/dark mode** toggle (saved to `localStorage`).

## Run locally

```bash
npx serve .
```

Open `http://localhost:3000` (or the port shown).

## Features

- Dark mode toggle in the header
- Marquee, stats counters, horizontal experiences strip
- Parallax, word-split hero, magnetic buttons, cursor glow, scroll progress bar

## Files

- `index.html` — structure
- `styles.css` — themes + layout
- `main.js` — GSAP + theme logic
